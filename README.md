在线太空射击游戏 - Unity WebGL版本

## 🎯 游戏介绍

这是一个基于Unity引擎开发的太空射击游戏，具有以下特色：

- 🛸 **多种敌人类型**：不同颜色的敌机有不同的行为模式
- 🔫 **武器系统**：单发、双发、追踪导弹等多种武器
- 🎮 **控制方式**：支持键盘、触摸和虚拟摇杆
- ✨ **视觉效果**：爆炸粒子、拖尾效果、屏幕震动
- 📱 **跨平台**：支持桌面和移动设备

## 🎮 操作说明

### 桌面端
- **移动**: WASD 键或方向键
- **射击**: 空格键
- **全屏**: 点击右下角全屏按钮

### 移动端
- **移动**: 触摸屏幕左侧或使用虚拟摇杆
- **射击**: 点击屏幕右侧
- **拖拽**: 直接拖拽飞机移动

## 🛠️ 技术信息

- **引擎**: Unity 6000.1.9f1
- **平台**: WebGL 2.0
- **压缩**: Gzip启用
- **兼容性**: 现代浏览器 (Chrome, Firefox, Safari, Edge)

## 📋 系统要求

- 支持WebGL的现代浏览器
- 建议使用Chrome或Firefox获得最佳性能
- 移动设备需要足够的内存运行WebGL内容
